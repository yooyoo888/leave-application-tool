🚀 快速開始（3 步驟）
═══════════════════════════════════════════════════════════════

步驟 1：安裝 Python 套件
-----------------------------------
開啟 CMD，進入此資料夾，執行：
  pip install -r requirements.txt

步驟 2：執行工具
-----------------------------------
方法 A：雙擊「請假小工具.bat」
方法 B：在 CMD 執行「python 請假小工具.py」

步驟 3：選擇功能
-----------------------------------
程式會顯示主選單：
  [1] 開始執行請假程式
  [2] 查看使用說明（會開啟 Manual.txt）
  [3] 退出程式

🎯 打包成獨立 EXE
═══════════════════════════════════════════════════════════════

如果想要在沒有 Python 的電腦上執行：

  1. 雙擊「打包成EXE.bat」
  2. 等待自動完成（約 2-5 分鐘）
  3. 在「發布版本」資料夾取得 .exe 檔案

詳細說明請參考「如何打包成EXE.txt」
